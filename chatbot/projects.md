### 1. Visual Assistance Framework for Blind and Visually Impaired
**Status:** Completed & Published  
**Publication:** Multimedia Tools and Applications (Springer Q2, 2024)  

#### Problem
Visually impaired individuals face significant challenges in navigating their environment and understanding their surroundings, requiring real-time assistance that is both accurate and accessible.

#### Approach
Developed a fog-cloud-based AI framework that provides real-time assistance to visually impaired individuals using computer vision and intelligent scene understanding. The system leverages fog computing for low-latency processing and cloud computing for complex analysis.

#### Technologies
- Python
- Computer Vision
- Fog Computing Architecture
- Cloud Computing
- Deep Learning
- Real-time Processing

#### Impact
Published in a Q2-ranked international journal. The system demonstrates practical applicability in real-world scenarios.

### 3. Autonomous Parking Space Detection System
**Status:** Completed  

#### Problem
Parking slot occupancy detection requires accurate and efficient object detection under real-world conditions with varying lighting, weather, and occlusion scenarios.

#### Approach
Evaluation and deployment of multiple state-of-the-art object detection models (YOLO-NAS, YOLOv8, and DETR) on parking lot imagery using standardized training and evaluation pipelines. Comprehensive analysis of latency-accuracy trade-offs.

#### Technologies
- Python
- PyTorch
- YOLO-NAS
- YOLOv8
- DETR (Detection Transformer)

#### Outcome
Improved mean average precision (mAP) for parking slot detection while characterizing latency-accuracy trade-offs across different model architectures. Provided deployment recommendations based on resource constraints.

### 4. Road Object Detection and Localization with Binocular Vision

### 5. Intelligent Transportation Systems
**Status:** Completed & Published  
**Publication:** International Journal of Intelligent Transportation Systems Research (2025)

#### Problem
Industrial vehicle access control requires efficient license plate recognition, but most OCR systems are optimized for Latin characters and struggle with non-Latin scripts like Bengali, Hindi, or Arabic.

#### Approach
Developed an energy-efficient, real-time license plate recognition system supporting multiple language scripts. Optimized for edge deployment with focus on accuracy, latency, and power consumption.

#### Technologies
- OCR (Optical Character Recognition)
- Computer Vision
- Edge Computing
- Industrial IoT
- Deep Learning

#### Innovation
"Handles multiple language scripts (Bengali, Hindi, etc.) which most systems don't support well. Required redesigning both the model pipeline and deployment strategy for edge devices."

#### Outcome
Published in an international journal. System demonstrates robust performance across different scripts with acceptable latency for industrial deployment.
